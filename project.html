    'CalTube'

    A youtube design inspired project that is a collection of some of my favorite songs and thier videos. This project features 3 collections. 1 collection of 'Female Artist', 1 collection of 'Male artist', and 1 collection of 'Group artist'.

    This project has a sorting design where all the featured artist can be sorted by collection catagory. All featured artist can also be viewed in an 'All Artist Category'

    All featured artist in this project has thier own styled page, that display information about the featured song/video for the artist. The featured artist page also displays four related videos with a mock comment sections that contains information about the for related videos.

    'Pokemon Card Collection'

    A Pokemon Card Collection. That allows you to capture your favorite pokemon and add them to your 'Favorites' collecion.

    In this project you will be able to sort your favorite pokemon cards in neumerical order based on their card's id number. Sort from lowest to highest, or highest to lowest, with the click of a button. You will also be able to see the total number of collected cards in the favorites section.

    Also includes a feature that disaplys the total amout of Pokemons with the same special abilities.
